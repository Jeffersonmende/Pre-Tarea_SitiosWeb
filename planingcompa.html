PROPIEDADES CSS

DISPLAY

La propiedad Display es la que nos ayuda a realizar como se comporta la visualización de un elemento en nuestra pagína dependiendo de los valores que le demos, los cuales explicaremos a continuación

INLINE

Inline nos permite ver un elemento de forma lineal, también hay que saber que cualquier propiedad de altura y ancho no tendra efecto con inline

BLOCK

Block sirve para trabajar elementos en forma de bloque el cual empieza en una nueva linea hasta el mayor ancho del contenedor.

INLINE-BLOCK

Este valor de la propiedad display es una unión de los dos valores anteriores, supone un elemento en forma lineal pero que considera las propiedades de altura y ancho
